Graphing tool. Make nodes and connect them with edges

  • Press [n] to add a new node.
  • [Click] on a node to select it.
  • Click another node to create an edge between it and the selected node.
  • Press [d] to delete the selected node
  • Click a new location to move the selected node
  • All edges are directed, going from white to pink.
